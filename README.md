# Zero-Era_Genesis
此项目为本人开发的 “合金工业” 的发行版本，但区别在于此模组机制相对平衡，而合金工业存在严重平衡性问题。也可以认为 “合金工业” 为适用性测试版本。

另外您需要知道该项目和 “合金工业” 基本不同，在一些方面上会存在大量问题，此模组计划添加战役，但因为各种原因而废弃。也许是第一次学开发 Mindustry 模组的问题吧，但该项目是一定会添加的。在我完善第一和第二个区块要用的东西后就会开始写星球了。

在将来发布版本后，若游玩时出现BUG请及时提交反馈。

本模组中文名为 “零度纪元：起源”。

除此之外由于本人英语并不适合编辑英文部分，所以更新日志等仓库页面的英文翻译均 ai 提供。
注意：此项目使用的是 “GPL-3.0” 。


[===========================]

This project is the release version of 'Alloy Industry' developed by me, but the difference is that the mechanics of this mod are relatively balanced, whereas Alloy Industry has serious balance issues. It can also be considered a usability test version of 'Alloy Industry'.

Additionally, you should know that this project is fundamentally different from 'Alloy Industry' and will encounter numerous issues in some aspects. This module was planned to add a campaign but was abandoned for various reasons. Perhaps it was due to my first experience developing a Mindustry module, but this project will definitely be added. I will start writing about the planet after I have improved the materials needed for the first and second blocks.

After future releases, please submit feedback promptly if you encounter any bugs while playing.

The Chinese name of this mod is '零度纪元：起源'.

Besides, since my English is not suitable for editing the English sections, the English translations of repository pages such as the update log are all provided by AI.
